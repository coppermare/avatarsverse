# Avatarverse - App

Next.js application for the Avatarverse avatar library.

**Planning & specs**: [projects/experiments/avatarverse](../projects/experiments/avatarverse/)

## Quick start

```bash
npm install
npm run dev
```

- **Playground**: http://localhost:3000
- **API**: http://localhost:3000/api/avatar/voxel?seed=abc&size=64

## Structure

- `app/` - Next.js pages and API routes
- `core/` - Avatar generation (seed, styles, registry)
- Planning docs live in `projects/experiments/avatarverse/docs/`
